# Команды

На данной странице вы найдёте полный список команд на сервере

## Взаимодействия

### Аккаунт

| Команда | Назначение |
|---------|------------|
| `/premium` | Для тех у кого лицензия, вход без пароля. |
| `/cp` | Сменить пароль уч. записи. |

### Игрок

| Команда | Назначение |
|---------|------------|
| `/skin set [ник]` | Поставить себе скин по нику. |
| `/skin url [ссылка] CLASSIC/SLIM` | Поставить себе скин по нику. |
| `/sit` | Сесть на блок под игроком. |
| `/lay` | Лечь на блоки рядом. |
| `/spin` | Крутиться как с зачарованием тягун на трезубце (только для Ветерков). |

## Чат

| Команда | Назначение |
|---------|------------|
| `![текст]` | Написать в глобальный чат всего сервера. |
| `/msg [ник]` | Написать личное сообщение игроку. |
| `/r [сообщение]` | Быстрый ответ игроку писавшему вам в ЛС. |

## Прочее

| Команда | Назначение |
|---------|------------|
| `/tps` | Посмотреть TPS сервера. |
| `/ping` | Посмотреть средний PING сервера. |
