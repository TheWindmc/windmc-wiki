# Интерактивные таблички

На табличках можно отображать предметы — удобно для витрин и декора.

![Интерактивные таблички](/assets/interactivesigns.png)

## Как использовать

Возьми предмет, который хочешь отобразить, и кликни `ПКМ` по табличке.

## Удаление предмета

Нажми `ЛКМ` по табличке с предметом, чтобы забрать его.
