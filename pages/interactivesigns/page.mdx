# Интерактивные таблички

На нашем сервере таблички получили уникальную функцию — теперь на них можно отображать предметы, создавая витрины и декоративные элементы.

![Интерактивные таблички](/assets/interactivesigns.png)

## Как использовать

Чтобы добавить предмет на табличку нужно взять предмет, который хотите отобразить и кликнуть `ПКМ` по табличке с предметом в руке.

## Удаление предмета

Чтобы забрать предмет с таблички нажмите `ЛКМ` по табличке с предметом.
