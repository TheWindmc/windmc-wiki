# Невидимые рамки

Этот плагин позволяет использовать невидимые рамки предметов для игроков в выживании.

## Использование

![Невидимые рамки](/invisibleframes.webp)

Плагин позволяет быстро сделать рамку невидимой без команд. В Minecraft 1.16 появилась функция, которая позволяет сделать обычную рамку прозрачной. Раньше это было возможно только с помощью команд, но теперь плагин решает эту задачу. Невидимые рамки помогут реалистично разместить предметы на стенах.
