# Бот Wind Sync

Бот автоматически синхронизирует ники и роли игроков в Discord с никами и ролями в игре.

Wind Sync создан для ваших серверов и устанавливается по ссылке, как любой другой бот.

Добавьте бота на Discord сервер вашей общины — он выдаст роли игрокам сервера и изменит ники, чтобы они совпадали с никами в игре.

Это пригодится, чтобы запретить писать тем, кто не играет на сервере. Проще узнавать игроков, если их ники синхронизированы с игрой.

## 1. Добавьте бота на сервер

Нажмите на ссылку ниже и пригласите бота на ваш сервер. У бота должны быть права для выдачи ролей и изменения никнеймов.

[Пригласить бота](https://discord.com/oauth2/authorize?client_id=1307436413234577458&permissions=8&integration_type=0&scope=bot+applications.commands)

## 2. Настройте бота

Используйте команду `/settings`, чтобы включить синхронизацию ников, ролей или банов.

## 3. Создайте роли для синхронизации

Создайте роли, которые будет синхронизировать бот.

Их можно назвать как угодно, поставить любой цвет и любые права.

Если вы не хотите синхронизировать роли, можете пропустить этот пункт.

Необязательно синхронизировать все роли — можно синхронизировать только нужные вам.

## 4. Сделайте роль бота выше синхронизируемых ролей

Перетащите роль Wind Sync выше ролей, которые будет выдавать бот. Если роль бота будет ниже других ролей, то бот не сможет выдать роли.

![Роль бота выше других ролей](/assets/11.png)

## 5. Укажите боту, какие роли синхронизировать

Напишите команду `/set-role <роль> @role`, чтобы указать боту, какие роли синхронизировать.

Например: `/set-role player @[Synced]` сделает так, чтобы игроки сервера получали роль @Игрок Windmc.

![Пример команды](/assets/sync_1.png)

Повторите это для всех ролей или не повторяйте — указывать все роли необязательно.

Чтобы сбросить роль, напишите команду `/reset-role <role>`.

<Alert type="warning">
**Обратите внимание**

Не забудьте включить синхронизацию ролей в настройках бота
</Alert>

## 6. Проверьте настройки бота

Напишите команду `/settings`, чтобы проверить настройки бота перед запуском синхронизации.

![Настройки](/assets/sync_2.png)

## 7. Синхронизуйте игроков, которые зашли на сервер до установки бота

Напишите команду `/everyone-sync`, чтобы синхронизировать всех участников сервера.

![Синхронизация игроков](/assets/11.png)

Сделать это нужно только один раз — новые пользователи будут синхронизироваться автоматически. Может занять от двух секунд до двух минут из-за лимитов Discord.

![Синхронизация завершена](/assets/11.png)

## 8. Готово

Поздравляю! Вы настроили бота. Теперь он будет синхронизировать всех участников сервера и обновлять ники и роли, если они поменяются на основном сервере.

## Дополнительные команды бота

`/sync <@user>` — синхронизирует одного конкретного пользователя
