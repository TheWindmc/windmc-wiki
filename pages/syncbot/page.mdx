# Бот Wind Sync

Бот автоматически синхронизирует ники и роли игроков в Discord с никами и ролями в игре.

Wind Sync устанавливается на твой сервер по ссылке, как любой другой бот.

Добавь бота на Discord-сервер своей общины — он выдаст роли игрокам и изменит ники, чтобы они совпадали с никами в игре. Удобно, чтобы запретить писать тем, кто не играет на сервере, и проще узнавать игроков по нику.

## 1. Добавь бота на сервер

Нажми на ссылку ниже и пригласи бота на свой сервер. У бота должны быть права для выдачи ролей и изменения никнеймов.

[Пригласить бота](https://discord.com/oauth2/authorize?client_id=1307436413234577458&permissions=8&integration_type=0&scope=bot+applications.commands)

## 2. Настрой бота

Используй команду `/settings`, чтобы включить синхронизацию ников, ролей или банов.

## 3. Создай роли для синхронизации

Создай роли, которые будет синхронизировать бот.

Их можно назвать как угодно, поставить любой цвет и любые права.

Если не хочешь синхронизировать роли — пропусти этот шаг.

Необязательно синхронизировать все роли — можно только нужные.

## 4. Сделай роль бота выше синхронизируемых ролей

Перетащи роль Wind Sync выше ролей, которые будет выдавать бот. Если роль бота будет ниже — бот не сможет выдавать роли.

## 5. Укажи боту, какие роли синхронизировать

Напиши команду `/set-role <роль> @role`, чтобы указать боту, какие роли синхронизировать.

Например: `/set-role player @[Synced]` — игроки сервера получат роль @Игрок Windmc.

![Пример команды](/assets/sync_1.png)

Повтори для всех нужных ролей. Указывать все роли необязательно.

Чтобы сбросить роль — напиши `/reset-role <role>`.

<Alert type="warning">
**Обратите внимание**

Не забудь включить синхронизацию ролей в настройках бота
</Alert>

## 6. Проверь настройки бота

Напиши `/settings`, чтобы проверить настройки перед запуском синхронизации.

![Настройки](/assets/sync_2.png)

## 7. Синхронизируй игроков, которые зашли до установки бота

Напиши `/everyone-sync`, чтобы синхронизировать всех участников сервера.

Сделать нужно только один раз — новые пользователи будут синхронизироваться автоматически. Может занять от двух секунд до двух минут из-за лимитов Discord.

## 8. Готово

Поздравляю! Бот настроен. Теперь он будет синхронизировать всех участников и обновлять ники и роли, если они поменяются на основном сервере.

## Дополнительные команды

`/sync <@user>` — синхронизирует одного конкретного пользователя
