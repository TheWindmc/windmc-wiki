# Пластинки

Создавай собственные музыкальные пластинки и записывай на них любую музыку. Доступно только для подписчиков "Ветерок".

## Запись на козий рог

Помимо стандартных пластинок, можно записывать короткие мелодии до 15 секунд на козий рог.

## Команды записи

| Команда | Назначение |
|---------|------------|
| `/disc burn <url> [название]` | Записать диск с кастомным звуком по URL. Держи музыкальный диск в руке |
| `/disc erase` | Вернуть диск в обычное состояние |
| `/disc search <запрос>` | Поиск треков на YouTube |
