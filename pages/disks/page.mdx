# Пластинки

На нашем сервере доступна уникальная механика создания собственных музыкальных пластинок, позволяющая игрокам записывать и воспроизводить custom-музыку прямо в игре. Эта премиум-функция доступна исключительно для подписчиков "Ветерок" и открывает безграничные возможности для создания уникального музыкального контента.

## Основные возможности

С помощью специального аддона для PlasmoVoice вы можете:

- Записывать собственную музыку на пластинки
- Делиться своими творениями с другими игроками
- Воспроизводить записи через проигрыватель

## Запись на козий 

Помимо стандартных пластинок, можно записывать короткие мелодии до 15 секунд на козий рог.

## Команды записи

| Команда | Назначение |
|---------|------------|
| `/disc burn <url> [название]` | Создайте диск с пользовательским звуком по URL-адресу. Вам нужно подержать в руке музыкальный диск. |
| `/disc erase` | Вернуть диск в обычное состояние |
| `/disc search <запрос>` | Поиск треков на YouTube |
