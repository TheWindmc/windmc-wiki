# Универсальный котел

Добавлена механика покраски котлов из Bedrock Edition — теперь вы можете окрашивать воду в котле любым красителем и использовать её для покраски предметов.

![Покраска воды в котле красителями](/assets/universal-cauldron_01.gif)

## Как покрасить воду

Просто используйте любой краситель на котле с водой. Цвет можно смешивать, добавляя разные красители.

![Процесс покраски воды красителем](/assets/universal-cauldron_02.gif)

## Сброс цвета

Чтобы очистить окрашенную воду и вернуть её к обычному состоянию, используйте комок глины на котле.
