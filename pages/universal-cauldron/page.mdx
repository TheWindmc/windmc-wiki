# Универсальный котел

Механика из Bedrock Edition: окрашивай воду в котле любым красителем и используй её для покраски предметов.

![Покраска воды в котле красителями](/assets/universal-cauldron_01.gif)

## Как покрасить воду

Используй любой краситель на котле с водой. Цвета можно смешивать.

![Процесс покраски воды красителем](/assets/universal-cauldron_02.gif)

## Сброс цвета

Используй комок глины на котле, чтобы вернуть воде обычный цвет.

## Покраска предметов

![Покраска кожаной брони в окрашенном котле](/assets/universal-cauldron_03.gif)

Используй окрашенную воду для изменения цвета кожаной брони и других предметов.
