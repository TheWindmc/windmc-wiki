# Редактирование предметов

Меняй название и описание (лор) любого предмета. Доступно только подписчикам "Ветерок".

## Цветовое оформление

Система поддерживает два типа цветовых кодов:
- Стандартные цвета Minecraft (используй символ `&` и код цвета)
- HEX-цвета в формате `&#RRGGBB` (например, `&#FF5500` для оранжевого)

**Обрати внимание:** коды форматирования (жирный `&l`, курсив `&o` и другие) не поддерживаются.

## Основные команды

| Команда | Описание |
|---------|----------|
| `/rename [название]` | Изменить название предмета |
| `/lore add [текст]` | Добавить новую строку в конец описания |
| `/lore set [номер] [текст]` | Изменить или добавить строку с указанным номером |
| `/lore remove [номер]` | Удалить строку с указанным номером |
| `/lore reset` | Полностью очистить описание предмета |

## Изменение названия предмета

```
/rename [новое название]
```

**Важно:** не ставь пробел между цветовым кодом и текстом.

**Пример:** `/rename &cОгненный &6меч`

## Работа с описанием предмета

### Добавление строки в конец описания

```
/lore add [текст]
```

### Изменение или добавление строки с указанным номером

```
/lore set [номер строки] [текст]
```

### Удаление строки

```
/lore remove [номер строки]
```

**Важно:** нумерация строк начинается с 1. Строки можно добавлять не по порядку.

![Пример редактирования лора](/assets/itemedit3.png)

## Предупреждение

Слишком длинное название или описание может вызвать проблемы:
- Клиент может перестать загружать мир и отключиться от сервера
- Сервер может принудительно отключить тебя

В таком случае администрация удалит проблемные предметы из твоего инвентаря. Пользуйся редактором разумно.
