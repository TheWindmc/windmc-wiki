# Редактирование предметов

Редактирование предметов в руке.

## Команды

| Команда | Назначение |
|---------|------------|
| `/item name [название]` | Изменить название предмета |
| `/item lore [строка] [текст]` | Добавить описание предмета |
| `/item lore clear` | Очистить описание предмета |
| `/item enchant [зачарование] [уровень]` | Зачаровать предмет |
| `/item enchant clear` | Убрать зачарования |
| `/item unbreakable` | Сделать предмет неломаемым |
| `/item breakable` | Сделать предмет ломаемым |
| `/item hideflags [флаги]` | Скрыть флаги предмета |
| `/item hideflags clear` | Показать все флаги |
| `/item color [цвет]` | Изменить цвет кожи |
| `/item color clear` | Убрать цвет кожи |
| `/item glow` | Добавить свечение |
| `/item unglow` | Убрать свечение |
| `/item repair` | Починить предмет |
| `/item damage [урон]` | Установить урон предмета |
| `/item rename [название]` | Переименовать предмет |
| `/item reset` | Сбросить все изменения |
