# Редактирование предметов (Item Edit)

Item Edit — это премиум-функция нашего сервера, доступная всем подписчикам "Ветерок". С её помощью Вы можете изменять название и описание (лор) любого предмета, создавая уникальные вещи для своих коллекций, магазинов или особых случаев.

## Цветовое оформление

Система поддерживает два типа цветовых кодов:
- Стандартные цвета Minecraft (используйте символ `&` и код цвета)
- HEX-цвета в формате `&#RRGGBB` (например, `&#FF5500` для оранжевого)

**Обратите внимание:** коды форматирования (жирный `&l`, курсив `&o` и другие) не поддерживаются.

## Основные команды

| Команда | Описание |
|---------|----------|
| `/ie rename [название]` | Изменить название предмета |
| `/ie lore add [текст]` | Добавить новую строку в конец описания |
| `/ie lore set [номер] [текст]` | Изменить или добавить строку с указанным номером |
| `/ie lore remove [номер]` | Удалить строку с указанным номером |
| `/ie lore reset` | Полностью очистить описание предмета |

## Изменение названия предмета

Чтобы изменить название предмета, используйте команду:

```
/ie rename [новое название]
```

**Важно:** При использовании цветовых кодов не ставьте пробел между кодом и текстом.

**Пример:** `/ie rename &cОгненный &6меч`

## Работа с описанием предмета

### Добавление строки в конец описания

```
/ie lore add [текст]
```

### Изменение или добавление строки с указанным номером

```
/ie lore set [номер строки] [текст]
```

### Удаление строки

```
/ie lore remove [номер строки]
```

**Важно:** Нумерация строк начинается с 1. Вы можете использовать строки не подряд.

## Предупреждение

Из-за технических ограничений Minecraft слишком большое количество символов в названии или описании предмета может вызвать проблемы:
- Клиент может перестать загружать мир и отключиться от сервера
- Сервер может принудительно отключить Вас

В таких случаях администрации придётся удалить проблемные предметы из Вашего инвентаря, чтобы Вы смогли снова войти на сервер. Пожалуйста, используйте функцию редактирования предметов разумно.
