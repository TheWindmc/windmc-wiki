# Невидимый свет

На нашем сервере Вы можете создавать и использовать невидимые источники света для украшения своих построек.

## Как создать невидимый свет

Для создания невидимого источника света используйте следующий рецепт:

![Рецепт создания невидимого света](/assets/craftlight.png)

## Особенности использования

Каждый источник невидимого света имеет определённый уровень яркости от 0 до 15. Вы можете увидеть этот уровень в верхнем левом углу предмета в Вашем инвентаре. Чем выше значение, тем ярче будет освещение вокруг блока.

После размещения в мире эти блоки становятся полностью невидимыми и не имеют физической формы, но продолжают излучать свет.

Полезные свойства невидимых источников света:
- Вы можете перемещать их с помощью поршней
- Наблюдатели могут обнаруживать их
- Они тонут в воде

Используйте невидимый свет для создания уникальных световых эффектов и атмосферного освещения в Ваших постройках!
