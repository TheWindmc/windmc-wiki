# Невидимый свет

Этот плагин позволяет крафтить невидимые блоки света.

## Крафт

![Крафт невидимого света](/craftlight.png)

## Использование

Каждый блок света, будь то отдельный блок или предмет, имеет свой уровень освещенности, который варьируется от 0 до 15. В инвентаре этот уровень отображается в верхнем левом углу. Чем выше значение, тем ярче выглядят блоки.

При размещении в виде блока, эти элементы не имеют физической формы и становятся полностью невидимыми.

Блоки света можно двигать с помощью поршня. Их легко обнаружить с помощью наблюдателя. Также они тонут в воде.
