# Быстрый бетон (не работает)

На нашем сервере Вы можете быстро превращать порошок бетона в затвердевший бетон с помощью специального плагина. Просто бросьте порошок в котёл с водой!

## Как это использовать

1. Установите котёл на любую твёрдую поверхность

![Установка котла](/assets/fast1.png)

2. Наполните его водой (используйте ведро с водой)

![Наполнение котла водой](/assets/fast2.png)

3. Бросьте в котёл любой порошок бетона

![Бросаем порошок бетона](/assets/fast3.png)

4. Готово! Порошок мгновенно превращается в затвердевший бетон

Этот метод значительно экономит Ваше время по сравнению с обычным способом затвердевания бетона. Теперь Вы можете использовать сэкономленное время для создания более красивых и интересных построек!
