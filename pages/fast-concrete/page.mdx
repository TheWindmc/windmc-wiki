# Быстрый бетон

Брось сухой бетон в котёл с водой — он мгновенно превращается в твёрдый бетон.

## Как использовать

1. Установи котёл на любую твёрдую поверхность

![Установка котла](/assets/fast1.png)

2. Наполни его водой (используй ведро с водой)

![Наполнение котла водой](/assets/fast2.png)

3. Брось в котёл сухой бетон

![Бросаем сухой бетон](/assets/fast3.png)

4. Готово — блок мгновенно становится твёрдым бетоном
