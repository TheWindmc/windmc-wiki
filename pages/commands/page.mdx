# Полезные команды

На этой странице вы найдёте полный список доступных команд на нашем сервере.

## Взаимодействия

### Управление аккаунтом

| Команда | Назначение |
|---------|------------|
| `/premium <Пароль> confirm` | Для владельцев лицензии — вход без пароля |
| `/cp` | Изменить пароль на сервере |

### Персонализация и действия

| Команда | Назначение |
|---------|------------|
| `/skin set [ник]` | Установить себе скин по нику игрока |
| `/skin url [ссылка] CLASSIC/SLIM` | Установить скин по ссылке (укажите тип модели) |
| `/sit playertoggle` | Включить или выключить возможность сидеть на голове игроков |
| `/sit` | Сесть на блок под вами |
| `/lay` | Лечь на ближайшие блоки |
| `/map hide/show` | Скрыть/показать себя на карте (доступно только для Ветерков) |
| `/spin` | Эффект вращения как с зачарованием "тягун" (доступно только для Ветерков) |
| `/spit` | Плюнуть (доступно только для Ветерков) |
| `/fart` | Пукнуть (доступно только для Ветерков) |

## Общение

| Команда | Назначение |
|---------|------------|
| `![текст]` | Написать сообщение в глобальный чат сервера |
| `/msg [ник]` | Отправить личное сообщение игроку |
| `/r [сообщение]` | Быстрый ответ на последнее личное сообщение |
| `/me, /gme [действие] (/do, /gdo)` | Выполнить РП действие |
| `/roll, /groll [число] (от 3 до 999999999)` | Получить случайное число |
| `/try, /gtry [действие]` | Попытаться выполнить РП действие |
| `^ping !^ping` | Показать свой пинг |
| `^inv !^inv` | Показать свой инвентарь |
| `^loc !^loc` | Показать координаты |
| `^item !^item` | Показать предмет в руке |
| `^ec !^ec` | Показать свой эндер-сундук |

## Техническая информация

| Команда | Назначение |
|---------|------------|
| `/tps` | Проверить текущий TPS сервера |
