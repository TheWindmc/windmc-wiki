# Полезные команды

На этой странице вы найдёте полный список доступных команд на нашем сервере.

## Основные

| Команда | Назначение |
|---------|------------|
| `/menu` | Открыть главное меню |
| `/bank` | Открыть меню банкира и вызвать банкира |
| `/tps` | Проверить текущий TPS сервера |
| `/ping` | Проверить свой пинг на сервере |

## Аккаунт

| Команда | Назначение |
|---------|------------|
| `/swarns list` | Просмотреть свои варны |
| `/skin set [ник]` | Установить себе скин по нику игрока |
| `/skin url [ссылка] CLASSIC/SLIM` | Установить скин по ссылке (укажите тип модели) |

## Персонализация предметов

> Доступно только для [Ветерков](/wiki/subs)

| Команда | Назначение |
|---------|------------|
| `/rename [название]` | Переименовать предмет в руке |
| `/lore [текст]` | Изменить описание предмета в руке |

Подробнее — [Редактор предметов](/wiki/itemedit).

## Действия и анимации

| Команда | Назначение |
|---------|------------|
| `/sit` | Сесть на блок под вами |
| `/sit playertoggle` | Включить или выключить возможность сидеть на голове игроков |
| `/lay` | Лечь на ближайшие блоки |
| `/spin` | Эффект вращения (только для Ветерков) |
| `/spit` | Плюнуть (только для Ветерков) |
| `/piss` | Ссать (только для Ветерков) |
| `/fart` | Пукнуть (только для Ветерков) |

## Карта

| Команда | Назначение |
|---------|------------|
| `/map hide` | Скрыть себя на онлайн-карте (только для Ветерков) |
| `/map show` | Показать себя на онлайн-карте (только для Ветерков) |

## Чат и личные сообщения

| Команда | Назначение |
|---------|------------|
| `/msg [ник] [сообщение]` | Отправить личное сообщение игроку |
| `/r [сообщение]` | Быстрый ответ на последнее личное сообщение |

Подробнее о всех возможностях чата читайте на странице [Чат](/wiki/chat).

## РП-команды

| Команда | Назначение |
|---------|------------|
| `/me [действие]` или `/do [действие]` | РП-действие в локальном чате |
| `/gme [действие]` или `/gdo [действие]` | РП-действие в глобальном чате |
| `/try [действие]` | Попытаться выполнить РП-действие (локальный чат) |
| `/gtry [действие]` | Попытаться выполнить РП-действие (глобальный чат) |
| `/roll [число]` | Случайное число от 3 до 999999999 (локальный чат) |
| `/groll [число]` | Случайное число от 3 до 999999999 (глобальный чат) |

## Подписка Ветерок

| Команда | Назначение |
|---------|------------|
| `/veterok` | Персональное меню с изменением роста и меню партиклов |
| `/scale` | Открыть меню с изменением роста (от 0.85 до 1.25) |
| `/pp` | Открыть меню с изменением партиклов ходьбы |
| `/co i` | Режим проверки истории действий блоков |
| `/co near` | Показать ближайшие действия с блоками |

Подробнее — [Подписка "Ветерок"](/wiki/subs).
