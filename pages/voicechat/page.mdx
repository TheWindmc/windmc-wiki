# Голосовой чат

На сервере используется Plasmo Voice. Серверная часть — плагин для Paper, клиентская — мод для Fabric. Голосовой чат не обязателен для игры.

## Создание собственной музыки

На сервере можно создавать свои музыкальные пластинки и продавать их другим игрокам.

[Подробнее](/wiki/disks)

## Новые иконки статуса

Можно различать значки отключения звука на клиенте и на сервере.

Новый значок Player Muted Audio показывает, что игрок отключил голосовой чат на своём клиенте.
