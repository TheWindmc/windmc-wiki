# Кастомные предметы

На сервере доступны уникальные предметы для рп и создания атмосферы. Каждый предмет имеет свои особенности и механики использования.

> **Также смотрите:** [Мелкие механики](/wiki/mechanics) для других улучшений игрового процесса

## Сигарета

![Сигарета](/assets/items/cigarette.png)

Сигарета — предмет для рп, который можно закурить.

### Использование

Возьмите зажигалку во вторую руку и нажмите `ПКМ`, чтобы закурить. При закуривании зажигалка получает урон. Во время курения появляются частицы дыма, а в интерфейсе отображается прогресс-бар использования.

После полного использования сигарета превращается в окурок, который выбрасывается перед игроком. Окурок нельзя подобрать и он исчезает через некоторое время.

Между закуриваниями действует кулдаун.

## Пачка сигарет

Пачка сигарет позволяет хранить до 16 сигарет.

![Пачка сигарет](/assets/items/cigarette_pack.png)

### Крафт пустой пачки сигарет

<CraftingRecipe
  type="shaped"
  pattern={[
    ["paper", "paper", "paper"],
    ["paper", null, "paper"],
    ["paper", "paper", "paper"]
  ]}
  result="cigarette_pack_empty"
/>

### Использование

Зажмите `Shift` и нажмите `ПКМ` по пачке, чтобы извлечь одну сигарету.

### Заполнение пачки

Чтобы добавить сигареты в пачку, возьмите сигареты в курсор и кликните по пачке в инвентаре. Максимальная вместимость — 16 сигарет.

## Наручники

![Наручники](/assets/items/handcuffs.png)

Наручники позволяют ограничить передвижение другого игрока для сотрудников Отряда Специального Назначения (ОСН).

### Использование

Нажмите `ПКМ` по игроку с наручниками в руке, чтобы надеть их. Для использования наручников нужны права `kleashplayers.use`, а для того, чтобы на вас можно было надеть наручники, нужны права `kleashplayers.leashable`.

После надевания наручников игрок не может двигаться, но может поворачивать голову. Игрок получает эффект замедления и невидимости.

### Снятие наручников

Чтобы снять наручники, нажмите `ПКМ` по закованному игроку с ножницами в руке. После снятия наручники возвращаются тому, кто их надел.

Наручники автоматически снимаются, если игрок, который их надел, покидает сервер.

## Дымовая шашка

![Дымовая шашка](/assets/items/smoke_bomb.png)

Дымовая шашка — тактический предмет, который даёт временную невидимость и скорость.

### Использование

Нажмите `F` (смена рук), чтобы активировать дымовую шашку.

При активации игрок получает:
- **Невидимость** на 30 секунд
- **Скорость** на 30 секунд
- Визуальный эффект дыма вокруг игрока

Дымовая шашка имеет кулдаун 2 минуты между использованиями.
