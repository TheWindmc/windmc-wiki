# Мембраны Фантомов

На нашем сервере доступна уникальная механика получения мембран фантомов через специальный крафт. Поскольку фантомы не появляются в нашем мире, мы добавили альтернативный способ получения этого ценного ресурса для создания зелий и ремонта элитр.
