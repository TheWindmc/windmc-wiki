# Чат

На сервере кастомный чат с локальным/глобальным режимами, модулями и РП-командами.

## Основы

По умолчанию сообщение отправляется в локальный чат радиусом 128 блоков. Чтобы написать в глобальный — используй `!` в начале сообщения.

## Модули чата

Модули — специальные команды, которые позволяют делиться информацией прямо в чате. Поддерживаются форматы `^loc` и `:loc:` — используй любой удобный.

| Модуль | Назначение |
|--------|------------|
| `^ping` / `:ping:` | Показать свой пинг |
| `^loc` / `:loc:` | Показать текущее местоположение |
| `^item` / `:item:` | Показать информацию о предмете в руках |
| `^inv` / `:inv:` | Показать свой инвентарь |
| `^ec` / `:ec:` | Показать свой эндер-сундук |

Добавь `!` в начале для отправки в глобальный чат: `!:loc:`, `!^loc` и т.д.

## Личные сообщения

| Команда | Назначение |
|---------|------------|
| `/msg [ник] [сообщение]` | Отправить личное сообщение игроку |
| `/r [сообщение]` | Быстрый ответ на последнее личное сообщение |

## РП команды

| Команда | Назначение |
|---------|------------|
| `/me [действие]` или `/do [действие]` | Выполнить РП действие в локальном чате |
| `/gme [действие]` или `/gdo [действие]` | Выполнить РП действие в глобальном чате |
| `/try [действие]` | Попытаться выполнить РП действие в локальном чате |
| `/gtry [действие]` | Попытаться выполнить РП действие в глобальном чате |
| `/roll [число]` или `/groll [число]` | Получить случайное число от 3 до 999999999 (локальный/глобальный) |

## Упоминания и пинги

Напиши ник игрока в чате — получатель увидит твоё сообщение, подсвеченное золотистым, и получит уведомление.

Также можно использовать `@Ник` для упоминания игрока.

## Настройки чата

Открой `/menu` и зайди в настройки чата. Там можно:
- Отключить глобальный и локальный чат
- Настроить уведомления
