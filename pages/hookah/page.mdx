# Кальян

На нашем сервере есть механика кальяна с эффектами зелий и дымом, которая отлично дополняет рп.

## Как использовать

Чтобы покурить кальян:

1. Установите варочную стойку
2. Добавьте в неё зелье с эффектом
3. Зажмите `Shift` и нажмите `ПКМ` по варочной стойке

При использовании кальяна вы получите эффекты из зелья, а также увидите частицы дыма.
