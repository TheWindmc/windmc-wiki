# Кальян

На сервере есть механика кальяна — получи эффекты зелий и частицы дыма через варочную стойку.

## Как использовать

Чтобы покурить кальян:

1. Установи варочную стойку
2. Добавь в неё зелье с эффектом
3. Зажми `Shift` и нажми `ПКМ` по варочной стойке

При использовании получишь эффекты из зелья и увидишь частицы дыма.
